<h2>Instruções de instalação</h2>
1. Acesse o link do drive. <br>
2. Baixe o arquivo no formato zip. <br>
3. Após descompactar a pasta acesse a pasta Build. <br>
4. Em seguida, abra o arquivo "Collected.exe"
